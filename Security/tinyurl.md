
#TinyURL

    Get full URL of compressed URL's
    
## mod url examples

    ### tinyurl.com add preview
    http://tinyurl.com/zn7xnzu
    http://preview.tinyurl.com/zn7xnzu
    
    ###bit.ly add +
    http://bit.ly/2lgPesi
    http://bit.ly/2lgPesi+
    
    ### goo.gl
    https://goo.gl/vLfoaW
    https://goo.gl/vLfoaW+
