# CMD Prompt

    Explains how to get rid of file path and customize further
    
## examples

    prompt $get         >
    prompt $T%g         CurrentTime>
    prompt $D$get       CurrentDate>
    prompt $D$T$get     CurrentDateCurrentTime>
    prompt              default
    prompt %cd%$g       default
    prompt $P$g         default
    prompt $P$_$g       default, next line >    
    prompt /?           help
